Projeto de Infraestrutura DevOps

Descrição do Projeto

Este projeto consiste na implementação de uma infraestrutura completa utilizando Vagrant e Ansible, com quatro máquinas virtuais interconectadas proporcionando serviços de rede, armazenamento, banco de dados e aplicação web.
